# 摘要

全幅切片图像（WSIs）展示了层次结构，诊断信息源于细胞形态、区域组织及整体情境。现有的计算病理学（CPath）多模态大型语言模型（MLLMs）通常将整个WSI压缩为单一嵌入，这妨碍了细致的语义锚定，也忽视了病理学家如何在不同尺度上综合证据。我们提出了MLLM-HWSI，这是一种层次化的WSI级别的MLLM，它在四个不同的尺度上将视觉特征与病理语言对齐，分别为将细胞视作单词、将图块视作短语、将区域视作句子及将WSI视作段落，以支持可解释的基于证据的推理。MLLM-HWSI将每个WSI分解为具有特定尺度的多尺度嵌入，并共同施加（i）层次对比目标和（ii）跨尺度一致性损失，保持从细胞到WSI的语义连贯性。我们计算诊断相关的图块，并通过轻量级的细胞-细胞注意力融合（CCAF）变换器将分割的细胞嵌入聚合为每个图块的紧凑细胞词元。投影后的多尺度词元与文本词元融合，并输入到经过指令微调的LLM中，用于开放式推理、视觉问答（VQA）、报告和标题生成任务。经过三个阶段的训练，MLLM-HWSI在六个CPath任务的13个WSI级基准测试中取得了新的最先进结果。通过将语言与多尺度视觉证据对齐，MLLM-HWSI提供了准确、可解释的输出，反映了诊断工作流程，推动了整体WSI理解的进步。代码可在：GitHub获取。

# 1. 引言

癌症的诊断和预后依赖于使用千兆像素全切片图像（WSIs）的组织病理学评估，这仍然是临床的金标准[13, 53, 54, 69, 75]。计算病理学（CPath）的兴起为加速诊断工作流程、提高可重复性以及通过定量分析组织学景观实现早期癌症检测开辟了新的可能性[20, 27, 61]。WSIs 本质上是层次性的，无论是在生物学上还是结构上，能够捕捉到在多种放大倍数和尺度下组织的完整空间组织（图 1）[15, 19, 30, 75]。这种层次组织反映了组织本身的架构，诊断线索在嵌套层级中浮现，从细胞形态到区域以及整体结构模式[9, 15, 30, 75]。在细胞层面上，WSIs 捕捉到了多样的形态特征，包括细胞核大小变化、细胞质纹理和有丝分裂活性，这些共同定义了病理学的词汇[3, 50, 55]。在区域层面上，这些细胞形成微结构，例如腺体、导管或实性巢，这些定义了组织组织的语法并承载了诊断意义[11, 51]。在全局 WSI 层面，多个区域汇聚成一个连贯的组织架构，显示肿瘤与正常区域之间的空间关系、邻近结构的侵犯及坏死现象[6, 11, 15, 67]。这种多尺度组织构成了组织病理学解释的生物基础，支撑着人类专家和计算模型对癌症的推理[6, 37]。专家病理学家将 WSI 视为一个多尺度的景观，而非静态图像[6, 11, 26, 32]。诊断推理通常以低放大倍数开始，逐步深入区域组织形态的检查，最终结束于细胞特征的观察[56, 57]。病理学家将 WSIs 解读为结构化叙事，其中组织结构提供上下文，区域定义语法，而细胞则定义词汇[8, 22, 40]。这个过程是双向的：全局背景影响局部检查，而局部发现又细化全局理解，直到形成一致的发现[11, 26]。

在CPath中，提出了包括Quilt-LLaVA、SlideChat、WSI-LLaVA、TITAN、PRISM和HistGen等多模态大语言模型（MLLMs），用于视觉问答（VQA）、形态推理和报告生成等广泛任务。最先进的MLLMs，如SlideChat和WSI-LLaVA，将补丁级嵌入聚合为与相应报告对齐的单一WSI级表示。尽管该聚合捕捉了更高层次的上下文，但忽视了WSIs的分层结构，导致细粒度空间语义的丧失。此外，现有模型忽略了专家病理学家的临床工作流程，他们整合了通过逐步放大和上下文推理获得的多尺度视觉线索。

在本研究中，我们通过引入一种层次化的WSI级多模态大语言模型（MLLM-HWSI），针对这些局限性进行了改进，以实现对WSI的全面理解，包括分析、检索、病理推理和报告生成（图1-2）。我们的方法通过将单个细胞解释为单词、小块视图解释为描述细胞邻域的短语、较大区域解释为描绘组织结构的句子，以及整个WSI解释为形成疾病连贯视觉叙事的段落，从而解码潜在的病理语言[18, 21, 62]。我们将WSI的层次结构与多个尺度的病理报告进行对齐，确保MLLM-HWSI模拟病理学家的标准诊断工作流程。通过将文本描述（例如，多形性细胞核、基质浸润）与其对应的视觉部分相结合，模型捕捉了专家诊断背后的组合推理。这种多尺度对齐增强了可解释性，使得预测具有生物基础和可解释性（图2）。MLLM-HWSI弥合了病理学家对组织级别解释与计算模型推理之间的差距。与仅依赖全球嵌入的SlideChat [17]、TITAN [23]和WSI-LLaVA [44]不同，我们的模型将每个WSI分解为多个语义尺度——细胞、小块视图、区域和全球WSI——并为每个尺度学习不同的表示（图1）。在细胞尺度上，分割出的细胞被嵌入以表示形态学和细胞质特征，并且采用轻量级视觉变换器（ViT）与细胞交叉嵌入融合（CCEF）模块高效整合细胞信息。在更高的尺度上，层次编码器提取表示局部组织结构和全局架构的块、区域和WSI级嵌入。语义块过滤模块进一步优化块级令牌。这些嵌入通过特定于尺度的视觉语言（VL）投影器投射到共享的多模态空间，并与对应的文本描述对齐。通过共同施加层次对齐和跨尺度一致性，MLLM-HWSI保持了局部细胞特征与全局结构模式之间的诊断关系。对齐的视觉令牌随后与文本令牌在LLM预训练过程中融合，实现多尺度、基于证据的推理。MLLM-HWSI通过层次对比对齐损失和跨尺度一致性损失进行优化，以维持跨空间层次的语义一致性。最终，融合的多尺度视觉和文本令牌预训练出一个能够进行多尺度解释推理的LLM，反映出病理学家如何将细节与背景整合为连贯的诊断。我们对提出的MLLM-HWSI模型在六个不同的WSI级CPath任务上进行了评估，包括零样本分类、检索、视觉问答、报告生成、图像说明和跨模态检索，使用了13个公开可用的数据集。与24个最先进的CPath模型相比，MLLM-HWSI取得了显著的性能提升，如图2所示。我们的主要贡献为：1. 我们引入了一种多尺度层次化的MLLM，它实现了与病理报告的细胞、区块、区域和WSI级对齐，从而实现了对WSI的统一多尺度理解和推理。2. 我们共同优化了层次对比对齐和跨尺度一致性损失，以保持尺度间的语义一致性，从而实现多尺度和基于证据的推理。3. 通过将视觉层次与病理报告统一，我们的模型在诊断准确性和泛化能力上比仅依赖全局信息的MLLM有了显著提升。

# 3. 提出的层次化 WSI MLLM

概述：在本研究中，我们提出了分层WSI级多模态大语言模型（MLLM-HWSI），这是一个用于CPath中WSI的多尺度视觉理解和语言对齐的统一框架。MLLM-HWSI旨在将病理报告的文本内容与WSI中具体的空间和形态特征对齐，从细粒度细胞形态到全局组织结构。通过对齐层次化的视觉-文本表示，MLLM-HWSI实现了可解释且连贯的诊断推理，类似于病理学家如何在层次化尺度中整合观察结果。MLLM-HWSI架构的概述如图3（A）所示。它采用层次化多编码器设计，以捕获四个层次的语义信息。在细胞尺度上，使用CellViT编码器进行细胞分割，并提取描述核形态的细胞级嵌入。三个额外的编码器处理补丁、区域和WSI级表示，以捕获逐渐更广泛的结构和上下文信息。为了高效处理WSI，我们引入了两个关键模块：语义补丁过滤（SPF）和细胞-细胞注意力融合（CCAF）。SPF根据与文本嵌入的余弦相似度移除同质补丁，并选择具有诊断意义的异质补丁，以进行多模态预训练。CCAF利用轻量级ViT在每个补丁内执行细胞嵌入间的交叉注意力，生成一个聚合的细胞令牌，以捕获细胞级形态。在每个层次上，生成的嵌入通过特定于尺度的VL投影器投影到共享的多模态空间，进而将视觉特征与病理报告中的相应文本语义对齐。MLLM-HWSI共同优化两个互补的目标：（1）层次对比对齐损失，增强文本和视觉特征在每个尺度上的跨模态对应关系；（2）跨尺度一致性损失，强制不同空间层次间的语义一致性和层次对齐。对于多模态推理，已对齐的多尺度嵌入与文本令牌融合并集成到大语言模型中，支持层次化的指令微调。在预训练过程中，VL投影器和多尺度编码器共同优化，实现端到端的尺度间VL对齐。

# 3.1. 千兆像素全切片图的分层分解

WSI 通常超过 $100,000 \times 100,000$ 像素，因此直接的端到端处理在计算上是不可行的。我们对 WSI 进行层次化分解，以高效捕捉细粒度的细胞形态和全局组织背景 [15]。这不仅减轻了处理挑战，还反映出病理学家的工作流程。

在我们的模型中，WSI $I$ 在 $20 \times$ 下被划分为不重叠的区域，$I = \{ R _ { i } \} _ { i = 1 } ^ { n _ { r } }$，$R _ { i } \ \in \ \mathbb { R } ^ { 4096 \times 4096 \times 3 }$，其中每个区域 $R _ { i }$ 保留了足够的中观背景，以捕捉组织模式。每个区域进一步细分为更小的补丁，$\{ P _ { i j } \} _ { j = 1 } ^ { n _ { p } }$，$P _ { i j } \ \in \ \mathbb { R } ^ { 256 \times 256 \times 3 }$。总共从 $9,642 ~ W S I s$ 中提取了 0.356M 个区域和 91.33M 个补丁。层级分解允许高效的多尺度特征提取，同时保持跨层级的空间对应关系。它还使得 MLLM-HWSI 能够整合来自 $\{ P _ { i j } \} _ { j = 1 } ^ { n _ { p } } \to \{ R _ { i } \} _ { i = 1 } ^ { n _ { r } } \to I$ 的信息，从而促进层级 VL 对齐。

# 3.2. 体系结构

所提出的MLLM-HWSI的整体架构包括五个关键组件（图3（A），具体为：（i）层次多尺度编码器，（ii）细胞间注意力融合（CCAF）模块，（iii）语义补丁过滤（SPF）机制，（iv）层次 $\mathbf { V } { \to } \mathbf { L }$ 对齐投影器，以及（v）大语言模型（LLM）。这些组件共同使得MLLM-HWSI能够进行稳健的多模态推理。

# 3.3. 层次多尺度编码器

层次编码器在四个空间层面（细胞、补丁、区域和WSI）中捕获WSI语义，反映专家病理学家的诊断推理过程。补丁级编码器：在补丁级别，使用CONCH编码器提取视觉嵌入，该编码器捕获细致的纹理和中观结构线索，如腺体形成和间质组织：$f _ { i j } = \mathcal { F } _ { \mathrm { C O N C H } } ( P _ { i j } )$，其中$f _ { i j } \in \mathbb { R } ^ { d _ { p } }$表示补丁$P _ { i j }$的表示。语义补丁过滤（SPF）：考虑到大量补丁$\{ P _ { i j } \} _ { j = 1 } ^ { n _ { p } }$，SPF在保留诊断多样性和报告相关性的补丁的同时，消除冗余和同质的补丁。对于每个区域$R _ { i }$，$\{ f _ { i j } \} _ { j = 1 } ^ { n _ { p } }$被标准化，并计算成对余弦相似度：

$$

$$

# 3.4. 分层对齐 $\mathbf { V } \to \mathbf { L }$ 投影器

为了将层次视觉特征与语言模型的潜在空间对齐，我们采用四个不同的 $\mathrm{V} \to \mathrm{L}$ 投影器，分别对应于每个尺度：细胞级 $(A_{c})$，补丁级 $(A_{p})$，区域级 $(A_{r})$，以及整体切片级 $(A_{\mathrm{WSI}})$。在每个级别上投影的特征表示为：$z_{c} = A_{c}(c_{ij})$，$z_{p} = A_{p}(f_{ij})$，$z_{r} = A_{r}(r_{i})$，$z_{\mathrm{WSI}} = A_{\mathrm{WSI}}(f_{\mathrm{WSI}})$。

# 3.5. 多模态大语言模型 (LLM)

将投影嵌入与标记化的文本指令嵌入 $\boldsymbol { z } _ { \mathrm { t e x t } } \in \mathbb { R } ^ { l \times d _ { t } }$ 连接，以形成最终的多模态输入序列：$Z = [ z _ { c } , z _ { p } , z _ { r } , z _ { \mathrm { W S I } } , z _ { \mathrm { t e x t } } ]$，然后输入到大语言模型（LLM）中。这种融合使得 MLLM-HWSI 能够联合推理细胞贴片区域 $ \mathrm { W S I }$ 和文本上下文，从而实现全面的诊断解释。我们选择 Qwen2.5-7B-Instruct [74] 作为主干 LLM，因其出色的推理和遵循指令的能力。

# 3.6. 训练策略

阶段 1：层次跨模态对齐：最近的最先进 CPath 模型将全球 WSI 嵌入与整个病理报告进行对齐 [17, 42]，这限制了细粒度的语义对齐并降低了视觉问答性能（图 2）。MLLM-HWSI 通过层次性对比和跨尺度一致性目标实现了多层次的视觉文本对齐，捕捉了病理报告的语言层次结构。本阶段利用 9,642 个 WSI-报告对 [44]，更新所有层次编码器（ViTcell-cell, $\mathcal { F } \mathrm { C O N C H }$ , $\mathrm { V i T r }$ , ViTWSI）和文本编码器，同时保持 VL 投影器和 LLM 权重不变。设病理报告的词元嵌入为 $\mathbf { T } = \{ t _ { 1 } , t _ { 2 } , \dots , t _ { M } \}$ ，特定于尺度的对比损失为：

$$
\mathcal { L } _ { s } = - \frac { 1 } { n _ { s } } \sum _ { i } \log \frac { \exp ( \sin ( z _ { s , i } , t _ { i } ) / \tau ) } { \sum _ { j } \exp ( \sin ( z _ { s , i } , t _ { j } ) / \tau ) } ,
$$

其中 $s \in \{ c , p , r \}$ 表示单元格级、补丁级和区域级，$n_{ s }$ 表示该级别的视觉词元数量，$\tau$ 是控制分布尖锐度的温度参数。每个 $t_{ j }$ 对应于病理报告中的第 $j^{ \mathrm {th } }$ 词元嵌入，在分母中作为对比负样本。对于WSI级别，我们使用类似的公式：

$$
\mathcal { L } _ { \mathrm { W S I } } = - \frac { 1 } { n _ { b } } \sum _ { b } \log \frac { \exp ( \sin ( z _ { \mathrm { W S I } , b } , t _ { r , b } ) / \tau ) } { \sum _ { l = 1 } ^ { n _ { b } } \exp ( \sin ( z _ { \mathrm { W S I } , b } , t _ { r , l } ) / \tau ) } ,
$$

其中 $n _ { b }$ 表示批量大小，$t _ { r , b }$ 表示与每个 WSI 相关的病理报告的文本嵌入。虽然 $\mathcal { L } _ { s }$ 和 ${ \mathcal { L } } _ { \mathrm { W S I } }$ 确保了在各个尺度上的局部对齐，但它们并未确保相邻层级之间的语义一致性（例如，补丁与区域，区域与 WSI），导致跨尺度的语义漂移。为了解决这个问题，我们引入了一种跨尺度一致性损失，通过鼓励从细粒度到粗粒度表征的平滑过渡来促进层级一致性：

$$

$$

# 4. 实验

训练与实施：第一阶段预训练使用了9,642对WSIcaption（报告）[44]，训练50个周期，学习率为$1 0 ^ { - 3 }$，批处理大小$n _ { b } \sim 6 4$，以及$\tau \ 0 . 0 2$。包括$\mathrm{V i T _ { c e l l - c e l l }}$、$\mathcal{F}_{\mathrm{C O N C H}}$、$\mathrm{V i T}_{r}$和$\mathrm{V i T}_{\mathrm{W S I}}$在内的所有编码器和文本编码器进行微调。$\mathrm{V i T _ { c e l l - c e l l }}$包含两个变换器模块，每个模块有两个自注意力头。在预训练期间，我们采用了Qwen2.5-7B-Instruct作为主干大型语言模型[74]。第二阶段，训练了两个层次的VL投影器，批处理大小设置为256。在第三阶段，我们使用WSI-Bench [44]，学习率为$2 \times 1 0 ^ { - 5 }$，批处理大小为128。我们采用LoRA（秩128，$\alpha = 2 5 6$），并利用DeepSpeed ZeRO-3进行分布式训练。所有实验在4台NVIDIA A100 80GB GPU上运行。

CPath 任务和数据集：MLLM-HWSI 在六个 WSIl 级别的任务上进行评估。对于分类（零样本和线性探测），我们使用 BRACS（7 类）[10]，UBC-Ocean（5 类）[7]，TCGA-OT（46 类）[23, 49]，EBRAINS（30 类）[58]，PANDA（6 类）[12]，以及 IMP-CRC（3 类）数据集 [52]。零样本视觉问答在 WSI-Bench（4,119 对）[43]，WSI-VQA（8,672 对）[14]，SlideBench-VQA（BCNB: 7,247 对）[17] 和 SlideBench-VQA（TCGA: 7,824 对）[17] 上评估。报告生成在 WSI-Bench（208 对 WSIreport）[44] 和 HistGen（700 对）[29] 上评估。WSI 检索使用 TCGA-OT [23, 49]，EBRAINS [58] 和 IMP-CRC [52]。跨模态检索在 TCGA 报告 [23, 72] 上测量，注释生成在 SlideBench [17] 上进行。

# 4.1. 评估指标和最先进比较

对于分类任务，我们使用加权的 $F _ { 1 }$ 和均衡准确率（BA）[16]；对于报告/标题生成，使用 ROUGE、BLEU-14 和 METEOR [17, 59]；对于视觉问答（VQA）准确率 [59]；对于跨模态检索，使用召回率 $@ \mathrm { K }$ [23]；对于 WSI 检索，使用前 $1 \%$ 准确率 [23]。

对于零-shot 分类和 WSI 检索，我们比较了 10 个最先进的 CPath VLMs：PLIP、PathCLIP、MI-Zero、CONCH、QuiltNet、CPLIP、MR-PLIP、PathGenCLIP、TITAN 和 KEP，以及 PRISM。我们使用了 CONCH 推荐的数据集特定提示。对于线性探测和弱监督设置，我们与 HIPT、TITAN、UNI、CTransPath、REMEDIS、CHIEF、DINOPath、Virchow、GigaPath 和 RudolfV 进行了比较。对于 VQA、报告/说明生成和跨模态检索，我们基准测试了通用的 LMMs——GPT-4V、LLaVA、Qwen-VL-Max 和 Gemini-Pro-Vision，以及 CPath 特定的 MLLMs：Quilt-LLaVA、SlideChat、WSI-LLaVA、PRISM、MedDr、LLaVA-Med、HistGen 和 PathGen-LLaVA。为了公平起见，我们使用官方代码，一致的测试拆分和相同的推理提示。

# 4.3. 主要结果

1. 零-shot WSI 分类：所提的 MLLM-HWSI 模型在五个外部和一个内部数据集上与 13 个最先进的 CPath VLM 和 MLLM 进行比较，评估指标为 BA（图 4 (a)）。MLLM-HWSI 模型实现了平均 $BA$ 为 $71.86\%$，超越 TITAN $(64.56\%)$ 和 WSI-LLaVA $(61.01\%)$ 分别 $7.30\%$ 和 $10.85\%$。这一持续的改进突显了层次多尺度对齐的有效性。 2. 线性探测评估：我们在六个数据集上使用线性探测和弱监督分类对 MLLM-HWSI 与 11 个最先进的仅视觉和 VL CPath 模型进行比较，评估指标为 BA（图 4(b)）。MLLM-HWSI 实现了平均 BA 为 $82.48\%$，超越 TITAN $(75.68\%)$ 和 UNI $(72.86\%)$ 分别 $6.80\%$ 和 $9.62\%$。这些结果强调了我们层次多尺度视觉表征在更具判别性的特征学习中的贡献。 3. WSI 检索：评估 MLLM-HWSI 在五个数据集上的零-shot 检索性能，使用前 $1\%$ 准确率（图 4(c)）。MLLM-HWSI 实现了平均性能为 $85.62\%$，超越 TITAN $(80.06\%)$ 和 CONCH $(73.74\%)$ 分别 $5.56\%$ 和 $11.88\%$。这些改进验证了层次多尺度表征对高精度 WSI 检索的益处。 4. WSI VQA：我们在四个 VQA 基准上评估 MLLM-HWSI，以评估其在形态学、临床和病理任务中的多尺度推理和诊断理解（表 3）。这些基准需要详细的细胞级分析和整体的 WSI 级解释，为多模态推理提供了严格的测试。MLLM-HWSI 一直超越一般用途的 MLLM 和病理特定模型。平均而言，它在 SlideBench-VQA (TCGA) 上实现 $89.60\%$ 的准确率，在 SlideBench-VQA (BCNB) 上为 $68.70\%$，在 WSI-Bench 上为 $97.90\%$，在 WSI-VQA 上为 $69.20\%$，超越所有先前的最先进结果。这些提升源于 MLLM-HWSI 的层次视觉表征、通过一致性正则化损失实现的跨尺度 VL 对齐，以及加强上下文感知临床推理的指令微调。 5. WSI 报告生成：我们使用 WSI-Bench 和 HisGen 数据集评估 MLLM-HWSI 在报告生成方面的表现，并与一般用途和 CPath 特定模型进行比较（表 4）。MLLM-HWSI 在所有指标上都实现了最佳性能。这些结果超越了所有先前的最先进模型，证明了 MLLM-HWSI 生成准确、临床连贯且具有形态意识的诊断报告的能力。性能提升源于层次视觉对齐和跨尺度一致性，捕捉了细粒度的形态和高级的诊断背景。 6. WSI 标题生成：在 SlideBench-Caption 数据集上进行标题生成时，MLLM-HWSI 在所有指标上都表现最好（表 4）。它实现了 BLEU-1/2/3/4 的 $46.20\%$、$32.40\%$、$26.70\%$ 和 $23.10\%$，ROUGE-$L = 36.70\%$ 和 ME-$T E O R = 62.70\%$，显著超越 WSI-LLaVA。这些结果突显了模型在生成简洁、具有形态意识和临床相关的标题方面的强大能力，忠实地总结了 WSI 级发现。 7. # 5. 结论

我们在CPath中提出了一种分层多模态LLM，通过跨WSI的多尺度视觉-语言对齐来增强关键任务（如视觉问答、图像描述和报告生成）中的诊断理解。它将WSI分解为一个分层表示，包括细胞、图块、区域和WSI层级的嵌入。每个层级通过集成在MLLM中的专用视觉-语言投影器与文本语义对齐，使得在空间尺度上进行多粒度推理成为可能。提出的优化目标结合了三个互补组成部分，包括跨模态对齐、分层特征空间一致性和指令微调，以增强诊断推理。针对六个CPath任务的综合实验表明，MLLM-HWSI始终超过最先进模型，验证了分层多尺度对齐和跨模态推理的有效性。通过将分层视觉理解与以语言驱动的推理统一起来，MLLM-HWSI为CPath中的可解释基础模型建立了一种新范式，为临床决策提供了潜在的帮助。未来，我们计划将MLLM-HWSI扩展到更广泛的多模态医疗集成，包括放射学、基因组学和临床记录，以在统一的医疗AI框架内实现整体、以患者为中心的推理。
