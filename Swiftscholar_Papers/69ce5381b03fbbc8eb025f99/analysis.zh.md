# 1. 论文基本信息
## 1.1. 标题
论文标题为《Lightweight Prompt-Guided CLIP Adaptation for Monocular Depth Estimation》，核心主题是：提出一种参数高效的轻量框架，通过提示引导的混合适配器（Mixture-of-Adapters, MoA）适配预训练CLIP模型，用于单目深度估计任务，平衡低训练成本和高几何预测精度。

## 1.5. 摘要
### 研究目的
解决视觉语言模型（VLM，如CLIP）迁移到单目深度估计任务时的两个核心痛点：一是全量微调需要大量计算资源和标注数据，二是零/少样本适配方法几何精度不足、输出缺乏细粒度结构信息。
### 核心方法
提出参数高效框架MoA-DepthCLIP：在预训练ViT-B/32主干网络中插入轻量混合适配器（MoA）模块，仅选择性微调主干的最后几层；同时引入全局语义上下文向量引导特征适配，结合深度分箱分类+直接回归的混合预测架构，使用包含几何约束的复合损失函数训练。
### 主要结果
在NYU Depth V2室内深度估计基准上，相比原始DepthCLIP基线，$\delta_1$准确率从0.390提升至0.745（提升91%），RMSE从1.176降至0.520（降低55.8%），且可训练参数仅占全主干的极小部分。
### 关键结论
轻量、提示引导的混合适配器是将VLM语义知识迁移到细粒度单目深度估计任务的高效方案，能以极低的训练成本获得极具竞争力的精度。

## 4.2. 核心方法详解
### 4.2.1. 混合适配器（MoA）模块

#### MoA内部结构
每个MoA模块包含三个核心组件：专家网络、门控网络、残差注入，内部结构如下图（原文Figure 2b）所示：

##### 1. 专家网络（Expert）
每个专家是两层瓶颈结构的MLP，参数极少，公式与原文完全一致：
$$
\mathrm { Expert } ( x ) = W _ { 2 } \sigma ( W _ { 1 } x ) ,
$$
符号解释：
- $x \in \mathbb{R}^d$：输入token的特征向量，$d=768$为ViT-B/32的隐藏层维度；
- $W_1 \in \mathbb{R}^{d \times d_b}$：第一层线性层权重，$d_b=64$为瓶颈维度，大幅压缩特征维度减少参数；
- $\sigma$：GELU激活函数；
- $W_2 \in \mathbb{R}^{d_b \times d}$：第二层线性层权重，将特征维度映射回原维度$d$。
  每个专家的参数量仅为$768 \times 64 + 64 \times 768 = 98304$，4个专家的总参数量不足40万，相比ViT-B/32的8600万总参数，增量仅为0.46%。
##### 2. 门控网络（Gating Network）
为每个输入token动态计算$K$个专家的路由权重，实现token级的动态适配，公式与原文完全一致：
$$
g _ { i } = \mathrm { softmax } \left( \frac { G ( x _ { i } ) } { \tau } \right) ,
$$
符号解释：
- $x_i$：第$i$个输入token的特征向量；
- $G(\cdot)$：两层MLP，维度变换为$768 \to 128 \to K$，激活函数为ReLU；
- $\tau=2.0$：温度超参数，控制softmax输出的平滑程度，温度越高权重分布越均匀；
- $g_i \in \mathbb{R}^K$：第$i$个token对应$K$个专家的权重，每个元素取值在0到1之间，总和为1。
  作者的创新点在于：没有采用AdaMix等方法的随机路由策略，而是训练和推理阶段都使用确定性的加权融合，避免随机路由带来的训练不稳定，更适合深度估计这类对输出一致性要求高的几何任务。
##### 3. 混合与残差注入
将$K$个专家的输出用门控权重加权求和，再通过残差连接加回原输入特征，保证训练稳定，不破坏CLIP预训练的特征，公式与原文完全一致：
$$
\tilde { x } _ { i } = x _ { i } + \sum _ { k = 1 } ^ { K } g _ { i , k } \mathrm { Expert } _ { k } ( x _ { i } ) ,
$$
符号解释：
- $g_{i,k}$：第$i$个token对应第$k$个专家的权重；
- $\mathrm{Expert}_k(x_i)$：第$k$个专家对$x_i$的输出；
- $x_i$：原输入token特征；
- $\tilde{x}_i$：适配后的输出token特征。
##### 4. 专家多样性监控
为了避免专家坍塌（门控网络仅将token路由到少数几个专家，浪费多专家的表征能力），作者没有引入额外的负载均衡损失，而是通过监控门控分布的熵来验证专家的有效利用，公式与原文完全一致：
$$
H ( g _ { i } ) = - \sum _ { k = 1 } ^ { K } g _ { i , k } \log g _ { i , k } .
$$
熵越高说明多个专家都被充分使用，熵接近0说明出现专家坍塌。作者通过实验验证，本方法的门控熵始终保持较高水平，不需要额外的负载均衡损失。
除了MoA模块，作者还选择性微调ViT的最后4层Transformer块，其余层全部冻结，进一步提升适配能力，总可训练参数仍然仅占全主干的10%左右。
### 4.2.2. 全局场景上下文融合
原始DepthCLIP仅使用像素级的手工深度提示（如“近”“远”），缺乏全局场景语义引导。作者构造了无参数的全局场景上下文向量：
1.  针对NYU Depth V2的室内场景，构造一批场景相关的文本提示，例如“a photo of a kitchen”“a photo of a bedroom”“a photo of a classroom”等；
2.  将这些提示输入冻结的CLIP文本编码器，得到每个提示的特征，L2归一化后做元素平均，得到全局上下文向量$\boldsymbol{c}$，代表通用的室内场景语义先验；
3.  将$\boldsymbol{c}$广播到与视觉特征相同的空间尺寸，与视觉特征在通道维度拼接，得到融合后的特征，为视觉特征提供全局语义引导。
    该策略不需要引入任何可训练参数，能稳定提升深度预测的场景一致性。
### 4.2.3. 深度分箱策略
作者没有采用DepthCLIP的10个手工粗分箱，也没有采用AdaBins的自适应分箱（计算成本高），而是通过 ablation 实验验证：128个固定线性分箱是最优配置，既比粗分箱的粒度更细，能输出更精确的深度值，又不需要自适应分箱的额外计算开销，平衡了精度和效率。
### 4.2.4. 混合预测头
融合后的特征输入两个并行的预测头：
1.  **分类头**：预测每个像素属于128个深度bin的概率，用每个bin的中心值乘以对应概率加权求和，得到分箱的粗深度图，训练稳定，擅长学习场景的整体布局；
2.  **回归头**：直接预测每个像素的连续深度值，擅长捕捉细粒度的局部几何细节。
    最终将两个头的输出加权融合，得到最终的深度图，同时兼顾训练稳定性和细粒度精度。
### 4.2.5. 复合损失函数
采用三个损失的加权和作为总损失，同时监督分类头和回归头，平衡粗布局和细粒度精度，总损失公式与原文完全一致：
$$
\mathcal { L } _ { \mathrm { total } } = \lambda _ { \mathrm { cls } } \mathcal { L } _ { \mathrm { cls } } + \lambda _ { \mathrm { reg } } \mathcal { L } _ { \mathrm { reg } } + \lambda _ { \mathrm { silog } } \mathcal { L } _ { \mathrm { silog } } ,
$$
权重设置：$\lambda_{cls}=1.0$，$\lambda_{reg}=1.0$，$\lambda_{silog}=0.5$。
三个损失的详细解释：
1.  **分类损失$\mathcal{L}_{cls}$**：逐像素交叉熵损失，将真实深度值分配到对应的bin作为标签，监督分类头的预测，提供稳定的粗布局监督信号。
2.  **L1回归损失$\mathcal{L}_{reg}$**：逐像素的绝对误差损失，公式为$\mathcal{L}_{reg} = \frac{1}{N} \sum_{i=1}^N |\hat{d}_i - d_i|$，其中$\hat{d}_i$是回归头预测的深度，$d_i$是真实深度，$N$是有效像素数，监督细粒度的局部几何精度。
3.  **尺度不变对数损失$\mathcal{L}_{silog}$**：解决单目深度估计固有的尺度和偏移歧义问题，对全局尺度变化鲁棒，公式与原文完全一致：
    $$
    \mathcal { L } _ { \mathrm { silog } } = \alpha \left( \mathrm { Var } ( g ) + \lambda \mathrm { Mean } ( g ) ^ { 2 } \right) ,
    $$
    符号解释：
    - $g_i = \log \hat{d}_i - \log d_i$：预测深度与真实深度的对数差；
    - $\mathrm{Var}(g)$：所有有效像素$g_i$的方差，惩罚深度的相对误差；
    - $\mathrm{Mean}(g)$：所有有效像素$g_i$的均值，惩罚全局尺度偏移；
    - $\lambda=0.85$，$\alpha=10.0$：固定超参数。

# 5. 实验设置
## 5.1. 数据集
实验使用**NYU Depth V2**室内深度估计基准数据集：
- 来源：由微软Kinect传感器采集的真实室内场景数据；
- 规模：包含249个不同的室内场景，官方划分1449张训练图像、654张测试图像，所有图像均有稠密的深度标注；
- 特点：场景涵盖厨房、卧室、客厅、办公室等常见室内场景，深度值范围为0~10米，是单目深度估计领域最常用的室内基准数据集，能有效验证模型在室内场景的深度预测能力；
- 选择理由：原始DepthCLIP的基线结果在NYU Depth V2上发布，使用该数据集可以直接与基线公平对比，同时室内场景的语义信息丰富，能充分发挥CLIP的语义优势。
## 5.2. 评估指标
论文使用6个行业通用的深度估计评估指标，每个指标的定义、公式、符号解释如下：
### 5.2.1. 阈值准确率$\delta_t$（$t=1,2,3$）
1.  **概念定义**：衡量预测深度与真实深度的比值在指定阈值范围内的像素比例，值越高精度越高。$\delta_1$、$\delta_2$、$\delta_3$分别对应阈值`1.25`、$1.25^2=1.5625$、$1.25^3=1.953$，其中$\delta_1$是最严格的指标，反映高精度预测的像素占比。
2.  **公式**：
    $$
    \delta_t = \frac{1}{N} \sum_{i=1}^N \mathbf{1} \left( \max\left( \frac{\hat{d}_i}{d_i}, \frac{d_i}{\hat{d}_i} \right) < 1.25^t \right) , \quad t=1,2,3
    $$
3.  **符号解释**：
    - $\mathbf{1}(\cdot)$：指示函数，括号内条件满足时取值为1，否则为0；
    - $\hat{d}_i$：第$i$个像素的预测深度值；
    - $d_i$：第$i$个像素的真实深度值；
    - $N$：有效像素总数（排除深度值缺失的像素）。
### 5.2.2. 绝对相对误差（AbsRel）
1.  **概念定义**：预测深度与真实深度的绝对差相对于真实深度的平均值，值越低精度越高，衡量全局的相对误差，对远距离像素的误差更敏感。
2.  **公式**：
    $$
    \mathrm{AbsRel} = \frac{1}{N} \sum_{i=1}^N \frac{|\hat{d}_i - d_i|}{d_i}
    $$
3.  **符号解释**：与上述定义一致。
### 5.2.3. 均方根误差（RMSE）
1.  **概念定义**：预测深度与真实深度的差的平方的平均值的平方根，值越低精度越高，对大误差的惩罚更重，反映全局的绝对误差。
2.  **公式**：
    $$
    \mathrm{RMSE} = \sqrt{ \frac{1}{N} \sum_{i=1}^N (\hat{d}_i - d_i)^2 }
    $$
### 5.2.4. 对数误差（log10）
1.  **概念定义**：预测深度与真实深度的以10为底的对数差的绝对值的平均值，值越低精度越高，对深度的尺度变化不敏感。
2.  **公式**：
    $$
    \mathrm{log10\ Error} = \frac{1}{N} \sum_{i=1}^N |\log_{10}\hat{d}_i - \log_{10}d_i|
    $$
## 5.3. 对比基线
论文选择两类基线做公平对比：
1.  **官方基线**：原始DepthCLIP，ResNet-50骨干，零样本，10个手工深度分箱，是VLM适配深度估计的开山工作，作为核心对比基准；
2.  **自构造消融基线**：逐步添加组件的中间基线，包括ViT-B/32基础基线、加复合损失的基线、加复合损失+MoA的基线，用于验证每个组件的贡献。

### 结果分析
1.  **组件贡献逐步验证**：
    - 仅将原始DepthCLIP的ResNet-50骨干替换为ViT-B/32，$\delta_1$从0.390提升到0.417，RMSE降低0.08，说明ViT的特征更适合深度估计任务；
    - 加入复合损失后，$\delta_1$大幅提升到0.503（提升20.6%），RMSE降到0.843，说明复合损失是低资源下提升精度的核心因素，贡献甚至高于MoA模块；
    - 加入MoA模块后，精度进一步小幅提升，验证了MoA的特征适配作用；
    - 将深度分箱从10提升到128后，$\delta_1$大幅提升到0.745（提升46.7%），RMSE降到0.520，说明分箱粒度对VLM适配的深度估计精度影响极大。
2.  **最终性能提升**：MoA-DepthCLIP相比原始DepthCLIP基线，$\delta_1$提升91%，RMSE降低55.8%，效果提升极为显著，达到了实用级的精度。
3.  **$\delta_3$小幅下降的解释**：最终模型的$\delta_3$从0.931降到0.895，作者解释是因为分箱变细后，模型的预测更偏向细粒度，少数模糊区域的预测会从“粗但安全”的区间落到$\delta_3$的宽松阈值之外，但更严格的$\delta_1$和$\delta_2$大幅提升，说明模型的高精度预测能力显著增强，是正向的 trade-off。

# 7. 总结与思考
## 7.1. 结论总结
本文提出的MoA-DepthCLIP是一种参数高效的CLIP适配单目深度估计框架，通过轻量混合适配器模块、选择性微调骨干最后几层、全局场景上下文引导、混合分类回归预测头、复合几何损失的组合设计，在NYU Depth V2基准上实现了远超DepthCLIP基线的性能：$\delta_1$提升91%，RMSE降低55.8%，且可训练参数仅占全主干的极小部分。该工作证明了轻量的参数高效微调策略可以有效弥合VLM语义表征与几何任务精度需求的Gap，为低资源下的稠密几何任务提供了新的解决方案。
## 7.2. 局限性与未来工作
### 作者指出的局限性
1.  目前仅在室内数据集NYU Depth V2上验证了效果，未扩展到户外场景（如KITTI自动驾驶数据集）；
2.  全局场景上下文向量是针对室内场景手工构造的，泛化性有限。
### 未来工作方向
1.  将框架扩展到户外深度估计数据集，验证跨场景的泛化能力；
2.  引入动态的注意力-based提示选择机制，替代手工构造的全局上下文向量，进一步提升性能和泛化性；
3.  探索少样本/零样本设置下的适配，进一步降低对深度标注的依赖；
4.  优化MoA模块的推理效率，适配边缘设备部署需求。
## 7.3. 个人启发与批判
### 启发
1.  **跨范式融合的价值**：将NLP领域的参数高效微调方法与计算机视觉领域的经典几何约束结合，往往能以极低的成本获得极大的性能提升，该思路可以迁移到语义分割、表面法线估计、3D检测等其他稠密预测任务；
2.  **基础组件的贡献大于复杂设计**：实验中复合损失和分箱粒度的贡献远高于MoA模块，说明在低资源场景下，优先优化基础的监督信号和任务建模方式，比追求复杂的模型结构收益更高。
### 潜在改进点
1.  **泛化性问题**：当前的全局上下文向量是针对NYU室内场景手工构造的，迁移到其他场景时需要重新设计提示，可以探索自动生成通用场景上下文向量的方法；
2.  **标注依赖问题**：当前仍然使用全监督训练，依赖NYU的全部深度标注，可以结合自监督几何约束，进一步降低标注需求，实现少样本/零样本适配；
3.  **推理效率问题**：MoA模块的门控和多专家计算会带来一定的推理延迟，可以通过专家合并、量化等方法优化推理速度，适配边缘设备部署。
